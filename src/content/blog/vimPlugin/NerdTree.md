关于 NerdTree 的一些快捷键用法

| 快捷键 |           作用           |
| :----: | :----------------------: |
|   go   |           预览           |
|   i    |     新水平分割的窗口     |
|   s    |     新竖直分割的窗口     |
|   t    | 打开文件到一个新的标签页 |
|   i    |  toggle 隐藏文件的显示   |
